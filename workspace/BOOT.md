# 网关重启清单 (Crash Recovery)

由于外部上下文截断或系统重载带来的重新连接，需遵循以下恢复协议：
1. **镇定与重建**：禁止向用户询问无意义的“上次进行到哪了”，主动查阅 `MEMORY.md` 最新指针、最近的 `task.md` 以及 Git Worktree `git status` 收集未提交的痕迹。
2. **溯源未竞事务**：排查 `[/]` 进行中状态的事务节点。
3. **安全确认闭环**：如果不确定远程状态，立即通过 `git fetch` 和分支对比核定自己所在的时间线进度，防止覆盖团队产出。
4. **恢复宣告**：以高度压缩的信息汇报断点恢复策略：“重启追踪到 XXX 相关变更，现继续执行...”。
