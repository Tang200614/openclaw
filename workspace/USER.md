# 用户档案与环境预置

## 当前环境上下文
- **OS**: Windows
- **File Encoding**: UTF-8
- **Package Manager**: npm / pnpm / yarn (须优先寻找项目中的 lock 文件进行判断)
- **Shell**: PowerShell (在 CLI 执行命令时需注意单双引号等转义兼容性与路径处理)

## 用户技术偏好
- **架构品味**：倾向于现代化的技术栈。对于新项目，注重结构清晰与高复用性设计。
- **审美底线**：不能容忍粗糙的“默认 HTML 风”。前端界面的交付必须达到生产级/企业级的现代感与统一感（排版、留白、字体、动效必须规范）。
- **执行偏好**：每次多子代理调度结束后，立即销毁 `c:\Users\Administrator\.openclaw\subagents\runs.json`，避免残留运行记录。

*(注：本档案将在后续长期对话中，根据用户指令和纠偏逐步扩展充实。)*
